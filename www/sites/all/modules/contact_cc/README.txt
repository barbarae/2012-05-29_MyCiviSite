Contact_CC
==========

Contact_cc provides CC and BCC recipient fields to the sitewide contact form.
These fields are configurable per category.


Dependencies
============

* Contact


Installation and Usage
======================

1) Copy the contact_cc folder to the modules folder in your installation.

2) Enable the module (/admin/modules).

3) Create or edit categories for your sitewide contact form
   using Structure -> Contact form (admin/structure/contact.
   In the category editing form, you will now have text areas for
   CC recipients and BCC recipients. In the same fashion as the
   core Recipients field, add a comma separated list of valid email 
   addresses to these fields.
